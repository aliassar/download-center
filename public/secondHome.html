what is khorshid ?
The purpose of Khorshid project is to make a personal page for each user. By getting each student's entry year, name of faculty, and username a personal page will be created for each user in a specific address so there will be the possibility of static website creation!!!

This project tries its best to help the users with their needs and has the ability to be expanded in future and present new features according to trending.
